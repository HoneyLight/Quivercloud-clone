How do you use Quiver?

In this demo you will learn how you can best use Quiver for your organization. We show you around so that you can quickly and easily work with Quiver. Within 5 minutes you will meet the highest security standard in the file sharing market.

These points are covered in this demo:

  1. Register for free
  2. My Space / own files library
  3. Spaces
  4. Files

Create a Quiver Account for free

Registering with Quiver is quick and easy. You can create a new account or log in with your Gmail or Microsoft account with the click of a button.

Creating a Quiver account is free. You have unlimited access to the Free package. And you get 30 days free access to the Quiver Pro functionalities.

Always work with the highest security standard on the market. Upgrade to Plus or Pro for the more extensive functionalities.

My Space

Congratulations, you have a Quiver account. From now on you can securely share information and much more.

On the Quiver platform you will find My Space, this is your personal library. You can also create a Space, more on that later.

There are three columns in My Space: 'MY FILES' , 'SHARED WITH ME' and 'SHARED WITH OTHERS'.

My Files

In MY FILES you will find all your secure uploads. You can easily upload your most used file types such as PDF, Word, Excel, PowerPoint and/or images. Create a folder to organize your files. Click share to share the file with others

Shared with me

In 'SHARED WITH ME' you will find all files that other Quiver users share with you. See at a glance who is sharing this file with you. The sender is 100% owner and determines the user rights for you as the recipient. Depending on your rights, you can open, download, edit, print, forward and/or digitally sign the file.

Shared with others

In 'SHARED WITH OTHERS' you will find all files that you share with other people. See at a glance who you're sharing this file with. As the sender, you are 100% owner and determine the recipient's user rights. Easily adjust the permissions: stop sharing, stop online editing, stop printing, stop forwarding and/or stop digital signing. After stopping sharing, the recipient will no longer be able to access the file, even after the recipient has downloaded the file.

How does Quiver spaces work?

On the Quiver platform you can create a Quiver Space, which is a secure data room for a project or collaboration with various (external) persons.

Create a new Space

Select Add space on the Spaces page to create a new Space.

  1. Enter the new space name.
  2. Consider making it a Private Space (Members will not be able to see other members or add files to this space).
  3. Enter and select the name or email address of the members you want to add.
  4. Click 'Add'

The creator of the Space is automatically Team Admin. Only a Team Admin can add, manage and/or designate Team Members as Team Admin.

In the Space, all Team Members can upload and share files. The person sharing the file has 100% ownership and determines user rights. After stopping sharing, Team Members will no longer be able to access the file, even after a Team Member has downloaded the file.

Files

Quiver means secure information sharing and much more. Other functionalities are:

Edit a file online You can create and upload new files on the Quiver platform. Quiver supports .docx, .xlsx and .pptx file types for online editing

Open or create a new file and choose 'Edit in Browser'. You now have access to known editing functions of the respective file types.

Monitor user activity of a file

Quiver users can monitor the user activity of their own files in the 'notification stream'. This is an audit trail in chronological order that shows who opens, closes, shares, forwards and/or downloads a file and when.

As owner you can also monitor at file level. In the file dashboard you will find 'Comments', 'Shared with' and 'Activity'.

  • Accepted file types

  • PDF
  • Microsoft PowerPoint (.pptx, .ppt)
  • Microsoft Word (.docx, .doc)
  • Microsoft Excel (.xlsx)
  • OpenOffice (ODF)
  • Images (bmp, gif, jpeg, jpg, png, svg, tiff, webp, heic)

  • Upload a file to 'My Space'

  • Go to 'My Space'.
  • Click 'Upload Files'.
  • Choose the files you want to add.
  • Click 'Upload'.

  • Upload a file to 'Spaces'

  • Go to the relevant 'Space'.
  • Click 'Add Files'.
  • Click on 'Select from My Files' or 'Upload'.
  • Choose the files you want to add.
  • Click 'Upload'.
